# Calapres Shopify Theme

Calapres (كالابريز) is an Arabic-first Shopify Liquid theme for a Riyadh-based niche perfume, luxury oud, and incense burner store.

## Branches

- `shopify-theme`: Shopify theme files only.
- `main`: sync and automation code in `sync/`.

Keep theme work out of `sync/`, and keep sync work out of the theme tree.

## Theme Tree

The production theme branch keeps the Shopify theme surface:

- `assets/`
- `config/`
- `layout/`
- `locales/`
- `sections/`
- `snippets/`
- `templates/`
- `design-source/`

`design-source/` is the committed Claude Design prototype source of truth for storefront structure, spacing, components, and assets.
